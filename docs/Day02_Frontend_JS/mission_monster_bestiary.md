## 🔧 Troubleshooting (デバッグ手法)
**「壊して学ぶ」** のがエンジニアの近道です。あえてコードを壊してみましょう。

1.  `app.js` を開き、`addEventListener` と書いてある行の先頭に `//` を打ち込んでください。
    ```javascript
    // button.addEventListener("click", function() {
    ```
2.  保存して、ブラウザをリロード（再読み込み）します。
3.  ボタンを押してください。

**結果**: 何も起きなくなりましたね？
**理由**: `//` は「コメントアウト（無視せよ）」という記号です。罠が無効化されたので、クリックしても反応しなくなったのです。

「動かない！」とパニックになった時は、こうやって「どこで反応が途切れているか」を探すのです。
