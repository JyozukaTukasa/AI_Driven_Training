# Drill 1: The Toggle Switch (スイッチ)

Day 2からは **JavaScript（動きの魔法）** です。
HTML/CSSで作った「ただの箱」に、命を吹き込みます。

最初のドリルは **「クリックしたら色が変わるスイッチ」** です。
スマホの「ダークモード切替」と同じ仕組みです。

## 🎯 Goal (完成イメージ)
-   画面にボタンが1つある。
-   ボタンを押すと、画面の背景色が「白」から「黒」に変わる。
-   もう一度押すと、「白」に戻る。

---

## Step 1: Block Sketch (ブロック分解)
「動き」を3つのステップで分解します。

1.  **トリガー (きっかけ)**: ボタンを「クリック」した時。
2.  **ロジック (判定)**: 今が「白」なら「黒」にする。逆なら戻す。
3.  **アクション (変化)**: 背景色を書き換える。

---

## Step 2: Visual Dictionary (辞書引き)
[Visual Dictionary](../../99_Resources/visual_dictionary.md) の「動き (Logic)」を見てみましょう。

-   **「クリックした時」** → `👆 イベント (.addEventListener)`
-   **「もし白なら」** → `⚖️ もし〜なら (if)`
-   **「背景色を変える」** → `🔨 アクション (function)`

---

## Step 3: Construct (構築)
AIに仕組みを伝えます。

### 🤖 Prompt Example
```text
【作りたいもの】
クリックすると背景色が切り替わる「ダークモードスイッチ」を作りたいです。

【構造 (HTML)】
- 画面の真ん中に `<button>` を1つ置いてください。
  - テキストは「切り替え」

【動き (JavaScript)】
ボタンが **クリック (Event)** されたら、以下の動きをしてください。
1. 今の背景色が「白」か「黒」かを判定する (if)。
2. 白なら黒に、黒なら白に入れ替える (Action)。
3. ボタンの文字も「Dark」⇔「Light」と切り替えてください。
```

---

## Step 4: Code Literacy (解剖)
生成された `<script>` タグの中身（JavaScript）を見てください。

1.  **「クリック」を見つける**
    -   `click` という単語はどこにありますか？
    -   `addEventListener` という長い呪文が見つかりますか？
2.  **「色」を決めている場所**
    -   `style.backgroundColor = "..."` のような行はありますか？
3.  **「実験」: 色を変える**
    -   コードの中の `black` を `red` に書き換えて、ボタンを押してみてください。
    -   画面が真っ赤になりましたか？

> **Why?**: プログラムは魔法ではなく、ただの「命令書」です。
> 「クリックしたら（いつ）」「色を変えろ（なにを）」が書いてある場所さえわかれば、自由自在に操れます。
