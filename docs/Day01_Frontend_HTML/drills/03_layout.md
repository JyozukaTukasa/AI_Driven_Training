# Drill 3: The Layout (骨格)

「部品（ボタン、カード）」の次は、**「画面全体（レイアウト）」** です。
Webサイトの9割は、この **「3段重ね（三種の神器）」** でできています。

1.  **ヘッダー (Header)**: ナビゲーションがあるアタマ。
2.  **メイン (Main)**: コンテンツがある胴体。
3.  **フッター (Footer)**: コピーライトがある足元。

## 🎯 Goal (完成イメージ)
一般的なWebサイトの骨格を作ります。

-   **ヘッダー**: 黒背景、左にロゴ、右にメニュー。
-   **メイン**: 白背景、真ん中に「ようこそ」。
-   **フッター**: 黒背景、「(c) 2024 My Site」。

---

## Step 1: Block Sketch (ブロック分解)
大きな箱を3つ積むイメージです。

**【構造メモ】**
-   全体（親箱）
    -   **縦積み**
-   1. ヘッダー（上の箱）
    -   **横並び**（ロゴとメニュー）
    -   両端に飛ばす
-   2. メイン（真ん中の箱）
    -   一番広いエリア
-   3. フッター（下の箱）
    -   真ん中に文字

---

## Step 2: Visual Dictionary (辞書引き)
[Visual Dictionary](../../99_Resources/visual_dictionary.md) を確認します。

-   **「アタマ」** の箱は？ → `<header>`
-   **「足元」** の箱は？ → `<footer>`
-   **「両端に飛ばす」** には？ → 「両端に配置して」

---

## Step 3: Construct (構築)

### 🤖 Prompt Example
```text
【作りたいもの】
一般的なWebサイトのレイアウト（骨組み）を作ってください。

【構造】
全体を「縦積み」にして、以下の3つを並べます。

1. ヘッダー (<header>)
   - 背景：黒、文字：白
   - 中身：「ロゴ」と「メニュー」を**横並び**にして、**両端**に離して置いてください。

2. メインエリア (<main>)
   - 背景：白
   - 画面の残りの高さを全部埋めてください (flex-grow: 1)。
   - 真ん中に「Welcome」と書いてください。

3. フッター (<footer>)
   - 背景：黒、文字：白
   - 真ん中に「(c) 2024」と書いてください。
```

---

## Step 4: Code Literacy (解剖)
コードを見て、**「構造の切れ目」** を探します。

1.  **アタマとカラダの境目は？**
    -   `</header>`（ヘッダーの終わり）と `<main>`（メインの始まり）を探してください。
2.  **「両端に飛ばす」魔法の正体は？**
    -   CSSの中に `justify-content: space-between;` という呪文はありますか？
    -   これこそが「要素を両端に引き離す」命令です。
3.  **「実験」: 色の反転**
    -   フッターの `background-color` を `white` に、`color` を `black` に変えてみてください。

> **Why?**: レイアウト崩れ（フッターが浮いてきちゃう問題など）を直すには、この**「箱の境目」**が見えるようになることが必須です。
