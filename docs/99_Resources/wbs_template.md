# 📅 Daily WBS Sheet (作業分解シート)

「今日、自分が何をするか」を言語化するためのシートです。
毎朝 09:00 にこのシートを記入（または更新）してから作業を開始してください。

## 1. The Goal (最終ゴール)
> 今のプロジェクト（またはDay）の完了条件は何か？

- [ ] 例: 「予約システム」をスマホで動く状態でデプロイする。

## 2. Weekly Milestones (今週のマイルストーン)
> ゴールに辿り着くために、今週（または今日）どこまで進むべきか？

- [ ] Step 1: 画面を作る (Frontend)
- [ ] Step 2: データを繋ぐ (Backend)
- [ ] Step 3: ...

## 3. Today's Scope (今日のスコープ)
> 今日1日（8時間）で終わらせるタスクを、**1時間単位** まで細かく分解する。
> ※「〇〇機能を作る (4時間)」はNG。「画面設計(1h)」「コーディング(2h)」「テスト(1h)」のように分解する。

| 時間 | タスク名 (Task) | 完了条件 (Definition of Done) | 予定 (Est) | 実績 (Act) |
|:---|:---|:---|:---|:---|
| 09:00 | **Morning Plan** | このシートを書き、今日のゴールを決める | 30min | |
| 10:00 | (例) ボタン実装 | ホバーで色が変わり、クリックできる | 1h | |
| 11:00 | ... | ... | ... | |
| ... | | | | |
| 17:00 | **Reflection** | 日報を書き、明日の予定を立てる | 30min | |

## 4. Risks (不安要素)
> 今日詰まりそうなポイントはどこか？（事前にAIに聞いておくべきこと）

- 例: Firebaseのセキュリティルールの書き方がわからない。
