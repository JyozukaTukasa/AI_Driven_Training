# AI-Driven Dev Training Kit (AI駆動開発新人研修マニュアル)

## プロジェクト概要
本プロジェクトは、プログラミング未経験者を10日間で「AIを活用して自走できるエンジニア」に育成するための研修マニュアルキットです。
「引き出し戦略 (The Drawer Strategy)」を軸に、暗記ではなく情報検索とAI活用による問題解決能力の習得を目指します。

## 主な特徴
- **The Drawer Strategy**: 知識のインデックス化による効率的な学習
- **Universal Engineering**: 特定の言語に依存しない普遍的な概念の理解
- **Guardian Manager**: AIをマネージャーとして活用した安全な進行

## 技術スタック
- **Frontend**: HTML/CSS/JS -> React
- **Backend**: Python (FastAPI)
- **Infra**: Firebase
- **Tools**: Cursor (AI Editor), Git

詳細については `GEMINI.md` を参照してください。
